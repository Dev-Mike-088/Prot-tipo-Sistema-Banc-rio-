# Prototipo-Sistema-Bancario #

Sistema Bancário 

Projeto simples em Java que simula um sistema bancário básico no terminal.

Funcionalidades

Ver saldo

Depositar dinheiro

Sacar dinheiro

Sair do sistema

Saldo inicial: R$ 1000

Como Executar 

javac Main.java
java Main

Objetivo

Treinar lógica de programação, métodos, Scanner, switch-case e laços de repetição em Java.
